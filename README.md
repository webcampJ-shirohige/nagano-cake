# 使用までの手順
$ git clone git@github.com:webcampJ-shirohige/nagano-cake.git<br>
$ bundle install<br>
$ rails db:migrate<br>
$ rails db:seed<br>
$ rails s<br>

# 機能
・アカウント作成なしで商品の閲覧が可能<br>
・商品をカートに入れ、1度に複数種類、複数商品の購入ができる。また、カート内の商品は個数変更・削除ができる<br>
・会員は配送先を複数登録しておくことが可能<br>
